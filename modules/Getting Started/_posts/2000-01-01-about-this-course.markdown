This course is designed to cover the very basics of using and working with R. It will be taught with the following sections:

- **Getting Started in R and RStudio**
- **Data in R**
- **The Tidyverse**
- **Graphing Data in R**
- **Modeling Data in R**
- **Communicating and Publishing Data in R**
